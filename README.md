hackd.mx
========

This is supposed to have setup information for the project.

[![Build Status](https://travis-ci.org/hackd/hackd.mx.png)](https://travis-ci.org/hackd/hackd.mx)

## Installation

```
$ git clone git@github.com:hackd/hackd.mx.git
$ bundle install
$ rake db:setup
```

## Run Application

```
$ rails server
```
